# 转场效果

目前使用的转场均来自于FFmpeg内置的 [Xfade](https://trac.ffmpeg.org/wiki/Xfade) 转场滤镜。

从模块引入转场效果常量即可使用。

```javascript
import { TRANSITION } from "web-video-creator";

console.log(TRANSITION.FADE);  // 输出 fade
console.log(TRANSITION.CIRCLE_OPEN);  // 输出 circleopen
```
